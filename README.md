# OpenCodeView

**Version: 0.7.4**

OpenCode is an AI coding assistant that provides intelligent code suggestions and automation. OpenCodeView, inspired by Shlomob's ocmonitor-share project, is a web-based monitoring tool that helps you track and analyze your OpenCode usage, including session history, token consumption, and cost estimation. This project is built with Bun, TypeScript, HTML, and CSS. Actually, almost all of the backend and frontend codes in this project has been generated and modified by the GLM-4.7 model and the OpenCode CLI.

## üìä Features

- **Session Analysis** - View detailed statistics of all OpenCode AI sessions
- **Cost Calculation** - Accurate cost calculation based on token usage
- **Time Reports** - Daily, weekly, monthly statistics
- **Model Analysis** - Statistics and costs by model
- **Project Analysis** - Development activity by project
- **Message Details** - View PRT files associated with each message
- **Modern UI** - Responsive design supporting mobile devices

## üöÄ Quick Start

### 1. Install Bun

If you haven't installed Bun yet:

```bash
curl -fsSL https://bun.sh/install | bash
```

### 2. Download OpenCodeView

```bash
# Clone the repository from GitHub
git clone https://github.com/huangli712/OpenCodeView.git
```

### 3. Install OpenCodeView

```bash
# Navigate to the project directory
cd OpenCodeView

# Execute the startup script
./start.sh
```

Or

```bash
# Navigate to the project directory
cd OpenCodeView

# Install dependences
bun install

# Development mode (with hot reload)
bun run dev
```

### 4. Access OpenCodeView

Open in your browser: **http://localhost:3000**
